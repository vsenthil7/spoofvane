Discovery 10 modules

How SpoofVane finds impersonation

SERP scanner

A2
Cert transparency
Tails CT logs (Calidog certstream) + Web Scraper API fallback for new lookalike certs.
A3
DNS brute
Permutation + bitsquatting + homoglyph + IDN; resolves via residential proxies.
A4
Registrar feed
Bright Data Datasets WHOIS feed; flags newly-registered & risky-country domains.
A5
Phishing feeds
OpenPhish + PhishTank + URLhaus + UrlAbuse ingest, normalised to one IOC schema.
A6
Social impersonation
X / IG / TikTok / LinkedIn / YouTube via SERP + Web Unlocker.
A7
URL shortener

A8

Crawl seed

Scoring 10 modules

How similarity & risk are quantified

C1
Composite scorer
Platt-calibrated probability (not a raw sum) with per-signal contribution breakdown.
C2
URL risk scorer
Entropy + suspicious tokens + TLD + punycode + IP + age → risk with contributions.
C3
DOM scorer
Structural tree-edit distance vs known-good login pages.
C4
pHash scorer
Hamming distance over perceptual hashes vs brand assets.
C5
Logo scorer
CLIP/MobileCLIP logo detection (feature-flagged, never 0% from missing weights).
C6
Kit fingerprinter
13 kit signatures (EvilProxy, Tycoon-2FA, Greatness, Mamba2FA…) via DOM + JS-bundle hash.
C7
Family classifier
ML classifier mapping kit features → threat family.
C8
Cluster score
Union-find threat-network clustering by shared ASN/registrar/kit/favicon/JA3.
C9
Voiceprint score
192-dim speaker embedding; cosine match against an enrolled executive voiceprint.
C10
Deepfake score
Ensemble: face embedding + lip-sync + C2PA provenance → deepfake probability.
